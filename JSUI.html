JSUI

What is it?

Previously on our site, we've used Dreamland JS to create our menu bar up at the top on our site instead of pasting code between html files.

So?

We made our own! And for its limited selection of items you can make with it, it works really well! In fact, the menu bar at the moment up top is made with JSUI

Feel like using it?

You sure? OK. Click the button below and the copied code should go at the bottom of your code

How to use JSUI

Create a P element: P(id, class, text, target)
Create a H element: H(type (1-6), id, class, text, target)
Create a Div: Div(id, class, target)
Create an A element: A(id, class, text, href, target)
Create a Button: Button(id, class, text, onclick, target)

If you don't want any element to have a certain quality like an onclick, id, class or text, just leave it blank
