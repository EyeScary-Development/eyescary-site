Stronge

NEWS!: Stronge 0.5 super early preview is out now! you can now add new lines before a line by typing newln : and the line number you want a newline after

Stronge is a minimal code editor written in Python

Features:

Write to any file type

if your code is in ESDlang it will work seamlessly with stronge, and is executable with our built in interpreter!

Can I download it?

sure, the latest completely stable preview is available below:

this is an older, fully stable version of stronge, features like changing previous lines are not in this version, this also uses an older version of esdlang. But these features can be found in stronge 0.3 and later

when you launch stronge, it will prompt you to come up with a filename, then a file extension, put your chosen file extension here or leave the input blank for .txt, then get to writing code! to clear a file, type filclr on a new line, to save a file and exit, type savex on a new line, and to run your ESDLang code, type savrun on a new line


looking for something new? how about Stronge 0.5 super early preview? using newln :(linenumber) you can now add new lines before any line, too!

new commands only in super early preview stronge versions:

formatting the ins command: type ins, space and then the line number you would like to amend, 2 colons will pop up and this is where you write your amended line

formatting the newln command: type newln, space and then the line number you want to add a new line after, you can also use newln : and the line number you want to create a new line before a line (mainly to be used for adding a newline before the current first line)
